# InsightForge AI - React Landing Page

Responsive single-page landing page built with Create React App for AI-powered business analytics SaaS targeting Indian SMEs. Converts visitors with clear value proposition, pricing CTA, and contact form.

## Features
- Hero section with compelling business-focused copy
- Three key feature cards (Smart Analytics, Data Security, Easy Integration)
- Two-tier pricing table (Starter free, Pro ₹499/month)
- Working contact form with state management
- Fully responsive mobile-first design
- Poppins font for modern SaaS aesthetic
- Smooth hover animations and transitions

## Tech Stack
- React 18 (functional components, hooks)
- Vanilla CSS (media queries, CSS Grid, Flexbox)
- Google Fonts (Poppins)
- Responsive breakpoints (768px mobile)

## Local Setup
```bash
npm install
npm start
```
